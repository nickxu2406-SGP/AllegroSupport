## 二、技术实现细节

### 邮件线程构建
```python
# 按 conversation_id 分组
threads = {}
for email in emails:
    conv_id = email['conversation_id']
    threads[conv_id].append(email)

# 按时间排序
for thread in threads.values():
    thread.sort(key=lambda x: x['sent_at'])
```

### 问答对提取逻辑
```python
# 识别外部邮件（问题）
external_emails = [e for e in thread if not is_internal(e['sender_email'])]

# 识别内部邮件（回复）
internal_emails = [e for e in thread if is_internal(e['sender_email'])]

# 提取问答对
question = external_emails[0]  # 第一个问题
answer = internal_emails[-1]   # 最后一个完整答案
```

### 邮件正文清洗
- 移除签名档（"Best regards", "此致"等）
- 移除联系方式（"电话：xxx"）
- 移除多余空行
- 保留核心问答内容
